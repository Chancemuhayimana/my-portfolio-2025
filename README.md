# Developer Portfolio (React + Vite + styled-components)

## Prerequisites
- Node.js 18+ and npm 9+

## Run locally
```bash
npm install
npm run dev
```
Then open the URL printed in your terminal (usually http://localhost:5173).

## Build for production
```bash
npm run build
npm run preview
```

## Customize
- Edit `src/App.jsx`: change your name, tagline, skills, projects, and contact text.
- Replace placeholder images with your own (put images in `public/` or use external URLs).
- Connect the contact form to EmailJS or Formspree for real email delivery.
